# News Blog

A news blog web application built using Node.js, Express, MongoDB, Mongoose, and EJS.

## Project Link

The project is deployed on render at: <[https://newsblog.herokuapp.com/](https://newsblog-0lp2.onrender.com)>

## Folder Structure

The folder structure of the project is as follows:

* `index.js`: The main application file.
* `controllers`: Folder containing controllers for user, category, and article management.
* `middleware`: Folder containing middleware functions for authentication, authorization, and validation.
* `models`: Folder containing Mongoose models for user, category, news, and article.
* `public`: Folder containing static assets such as CSS, images, and JavaScript files.
* `routes`: Folder containing routes for admin and frontend.
* `utils`: Folder containing utility functions such as error handling.
* `views`: Folder containing EJS templates for the application.
* `package.json`: The package file for the project.
